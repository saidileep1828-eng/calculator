# Calculator CLI App

## Objective

This project is a simple Command Line Calculator built using Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Exit Option

## Technologies Used

- Python 3
- VS Code

## Concepts Used

- Functions
- Loops
- Conditional Statements
- User Input

## How to Run

Open terminal and execute:

```bash
python calculator.py
```
